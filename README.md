# Company Employees Database (SQL Project)

## Overview

This project is a beginner-friendly SQL database project based on an Employee Management System. It demonstrates how to create and manage a relational database using MySQL. The project includes database creation, table creation, data insertion, data manipulation, and advanced SQL queries for analyzing employee information.

## Features

* Created an Employee Database using MySQL
* Designed Employee and Department tables
* Inserted sample employee and department records
* Performed CRUD operations (Create, Read, Update, Delete)
* Used filtering with **WHERE**, **BETWEEN**, **LIKE**, and **IN**
* Applied sorting using **ORDER BY**
* Used aggregate functions such as **COUNT()**, **SUM()**, **AVG()**, and **MAX()**
* Implemented **GROUP BY** and **HAVING** for data analysis
* Wrote subqueries to find highest-paid employees and compare salaries
* Used SQL Window Functions (**RANK()** and **DENSE_RANK()**) for employee ranking
* Demonstrated basic table modification using **ALTER TABLE**

## SQL Concepts Covered

* DDL (CREATE, ALTER, DROP)
* DML (INSERT, UPDATE, DELETE)
* DQL (SELECT)
* Filtering and Sorting
* Aggregate Functions
* GROUP BY & HAVING
* Subqueries
* Window Functions
* Basic Database Design

## Technologies Used

* MySQL
* SQL
